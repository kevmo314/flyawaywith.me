flyawaywith.me
====

Input
----

* people:
	* per path edge

* locations:
	* list of nodes

* times:
	* arrival OR departure time per node, OR
	* total time from start to end

Output
----

* cost
* node list of nearby layovers
* duration within each ...
* import / export query

* [sabre REST](https://developer.sabre.com/docs/read/REST_APIs)
